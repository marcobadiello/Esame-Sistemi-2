# Progetto esame Sistemi di elaborazione 2

## Descrizione generale
Questo progetto si occupa di prendere i dati di Spotify e fornirne una rappresentazione grafica.

## Per avviare
Scrivere nel CMD:
```bash
streamlit run app.py
```

## **TOP**
Questa pagina permette di mostrare la top delle canzoni più ascoltate in un periodo di tempo selezionato, di default il periodo selezionato è ytd ma a scelta si può cambiare anno oppure selezionare un periodo personalizzato
